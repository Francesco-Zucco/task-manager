# Task Manager App

A clean and minimal task management app built with Next.js and Tailwind CSS.

## Features

- Add and delete tasks
- Mark tasks as completed
- Filter tasks by category (work, personal, urgent)
- Filter tasks by status (all / completed / remaining)
- Local storage
- Responsive and basic UI

## Tech

- Next.js
- React
- Tailwind CSS
- shadcn for drawer

## Run the development server:

npm run dev
